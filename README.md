# Blog Application with GenAI Suggestions

A full-stack Blog application built with React (Vite), Node.js (Express), SQLite, and Google Gemini LLM. In this app, the users can write, edit, delete the blog posts. Additionally, integrate Google Gemini LLM to suggest the related blog topics or generate introductory paragraphs when users start writing a blog.

## Tech Stack
- **Frontend:** React.js (Vite)
- **Backend:** Node.js (Express)
- **Database:** SQLite
- **GenAI:** Google Gemini
